# Specification

## Summary
**Goal:** Rebrand the donation site for True Hope Foundation, replacing Gaza-specific branding with humanitarian relief messaging and updating visual assets.

**Planned changes:**
- Update all landing page copy, headings, and descriptive text to reflect True Hope Foundation's mission and truehopefoundation.in domain
- Replace hero banner with new True Hope Foundation humanitarian relief image
- Update campaign configuration with appropriate fundraising goal for True Hope Foundation

**User-visible outcome:** Users will see a fully rebranded donation site for True Hope Foundation with updated mission messaging, new hero banner, and campaign goals appropriate for humanitarian relief work.
